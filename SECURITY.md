# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | yes       |

## Reporting a Vulnerability

Please do **not** file a public GitHub issue for security vulnerabilities.

Open a [GitHub Security Advisory](https://github.com/KrakoX/capsule/security/advisories/new) instead, or email the maintainer directly (see the GitHub profile).

Include the following in your report:
- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

You will receive a response within 72 hours.
